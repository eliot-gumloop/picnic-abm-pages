## Bullet & List Icons (main icon set)

> **The primary icon set for bullets, lists, and inline marks.** Outline versions of the brand character shapes — no eyes, stroke-only, for use as functional bullet points and list markers.

### Blob bullet icons
- What it is: A set of **5 outline blob shapes** used as the main icons for bullets, list items, and inline marks. Stroke-only (no fill), rounded, hand-drawn quality.
- The 5 shapes:
  | # | Shape | Description |
  |---|-------|-------------|
  | 1 | Clover / 4-petal flower | Four-petal rounded flower blob |
  | 2 | Triangle | Soft triangle with rounded corners |
  | 3 | Rounded square | Square with generous rounded corners |
  | 4 | Oval / egg | Simple organic oval |
  | 5 | Dome / arch | Rounded-top arch shape |
- File: `/home/user/skills/brand-canon/assets/icons/blob-bullets.svg`
- Use it for: **Bullet points, numbered/unnumbered lists, feature callouts, inline marks, step markers.** These are the default bullet icons across brand surfaces (decks, docs, web).
- How to use: Cycle through the 5 shapes for variety in a list, or pick one consistently per context. Can be filled with brand colors or left as outline (black/`#191818` stroke).
- Never: Replace with generic bullet dots or off-brand icon libraries. Distort the stroke weight.
- Status: 🔒 · Source: Blobs.svg (June 2026 / Mahnoor)
