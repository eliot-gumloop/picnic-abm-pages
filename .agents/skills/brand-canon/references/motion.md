## Motion Usage Rules

- 🔒 The dot/halftone grid is the core motion language — animations are built from the same pixel-grid texture as the gradient backgrounds
- 🔒 Keep motion icons square (262×262 source) — scale proportionally, never distort
- 🔒 Loop cleanly — these are designed as seamless loops
- 🧪 Pair with matching concept copy (Build / Control / Collab / Optimize)
- 🧪 Use one motion icon per section — restraint over decoration
